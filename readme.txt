Aeropress

A program that keeps your computer from falling asleep by simulating a key press periodically

Inspired by Caffiene
https://www.zhornsoftware.co.uk/caffeine/

System tray functionality created using https://github.com/Infinidat/infi.systray
Aeropress icon https://www.flaticon.com/free-icons/aeropress
Keystrokes simulated using PyAutoGUI from https://github.com/asweigart/pyautogui/blob/master/docs/index.rst
